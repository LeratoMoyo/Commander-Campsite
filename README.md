App is used as a checklist for campers.
It allows them to input all the things they need for camp.
It also helps them calculate how many things they are at a time.
The main screen loads after 3 seconds of opening it and takes you to the next screen.
The main screen has a button to add more things or remove.
The main screen follows the ambience of camping.
The view screen shows a table which details a list of things available and their categories.
It also has a back button which allows them to go back.
Various data is stored using parallel arrays.
The stored data entails Item name e.g Sleeping bags; Category e.g Shelter; Quantity and Comments.
